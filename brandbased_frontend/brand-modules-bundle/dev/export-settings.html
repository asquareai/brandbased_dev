BrandBased — Settings export

file://

Reads everything that the Brand Console (Theme Styles, Theme Settings, Products, Upload Logo) has persisted to localStorage for this brand, on this browser, then bundles it into one JSON file and one PHP file your backend can ingest. Open this page in the same browser session as the dashboard so it can see the same data.

Brand Verification — preview state

Flips both Premium and Freemium Brand Verification pages between the normal "Verified" state and the red "Brand Not Verified" failure state. Useful for showing the design team / backend dev what users see when verification can't be completed.

Brands page — Plan tier preview

View whether this brand is using a Free or Premium plan. When Show is on, each brand card on the Brands page displays a small Premium pill on the top-left of its orb. Useful for previewing how the page will read once plan-tier data is wired up.

Brand Console — dashboard tier preview

Premium (default): the dashboard iframe behaves normally. Freemium mode: while brand-console-dashboard.html is open in another tab, every module except URLs under …/freemium/ is covered by the upgrade message so you can preview the paywall in-context.

Stripe purchase verify — preview modal

Opens the BrandBased "Stripe purchase verify" modal on the dashboard (asks for email + confirm email before handing off to Stripe). Same glass styling family as Account → Preferences.
