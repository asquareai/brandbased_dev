Standalone bundle. Uses bb-shared-ui.css, premium-bb-mobile-popup-chrome.css, and bb-smart-ui.js in this folder (not the HTML alone). Mobile popup rules apply below 840px width — narrow the window or use devtools device mode. Hard-refresh (Cmd+Shift+R) if styles/JS look stale on file://.

Run API + local server 
cd premium-bb-modal-standalone
python3 -m venv .venv && . .venv/bin/activate && pip install -r requirements.txt
uvicorn bb_smart_sizing_server:app --port 8001
python3 -m http.server 8090
# http://localhost:8090/Premium-BB-Modal.html

Other modules live in the parent ui-demo repo (links use ../).
